# Thumbs Up

Solana Based, React Voting App

# Install

## Setup
If this is the first time you are installing the project,
you need to initialize the base account of the solana program by generating a keypair
```
node createKeyPair.js
```

## Dependencies
```
npm install
```

## Run
```
npm start
```
